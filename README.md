# Online Chat 
 
## Описание проекта

Онлайн чат с возможностью залогиниться под гугл-аккаунтом

## Технологии

React, Less и firebase

## Запуск проекта

1. Cклонировать репозиторий

```bash
git clone
```
2. Установить все зависимости

```bash
npm install
```

3. Запустить проект 

```bash
npm run start
```
